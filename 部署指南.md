# 图书信息管理系统 — Render 一键部署教程

> 前后端合一，免费部署，手机电脑都能访问。

---

## 一、准备工作（5 分钟）

### 1. 创建 GitHub 仓库

1. 打开 [github.com](https://github.com) 并登录
2. 点击 **New repository**
3. 填写：
   - Repository name：`book-system`（任意名字）
   - 勾选 **Public**
   - 点击 **Create repository**

### 2. 推送代码到 GitHub

在你的电脑上打开 **PowerShell**，依次执行：

```powershell
cd "D:\桌面\所有作业\实验报告\web\work.4"

# 初始化 git（如果还没有的话）
git init

# 添加所有文件
git add .

# 提交
git commit -m "初版：图书信息管理系统"

# 关联你的 GitHub 仓库（把下面的 URL 换成你自己的）
git remote add origin https://github.com/你的用户名/book-system.git

# 推送（第一次需要加 -u）
git push -u origin main
```

> 💡 如果提示 `branch 'main' does not exist`，把最后一行改成 `git push -u origin master`

## 二、部署到 Render（免费，3 分钟）

### 1. 注册 Render

1. 打开 [render.com](https://render.com)
2. 点击 **Sign Up** → 用 **GitHub 账号** 一键登录
3. **不需要信用卡**

### 2. 创建 Web Service

1. 点击 **New** → **Web Service**
2. 选择 **Connect a Git Repository** → 找到你的 `book-system` 仓库 → 点击 **Connect**
3. 填写配置：

| 配置项 | 填写内容 |
|--------|---------|
| **Name** | `book-system`（任意） |
| **Region** | 选 Singapore 或 Frankfurt（离中国近） |
| **Root Directory** | 留空 |
| **Runtime** | Node |
| **Build Command** | `cd book-server && npm install && cd ../book-front && npm install && npm run build && cd ../book-server` |
| **Start Command** | `node app.js` |
| **Instance Type** | **Free** |

4. 点击 **Advanced**，添加环境变量：

| Variable | Value |
|----------|-------|
| `PORT` | `3000` |

5. 点击 **Create Web Service**

### 3. 等待部署完成

大约 2-4 分钟，状态会从 **Provisioning** → **Building** → **Live**。

绿色 **Live** 出现后，点击顶部的域名链接（类似 `https://book-system-xxxx.onrender.com`），就能访问你的网站了！

## 四、常见问题

### Q1：网站打不开 / 502 错误

- 等待 2-3 分钟，Build 需要时间
- 点击 **Logs** 标签查看是否有报错
- 确保环境变量 `PORT = 3000` 已设置

### Q2：访问慢 / 需要等很久

- Render 免费版不活跃后会暂停，下次访问需要 30-60 秒唤醒
- 点一下页面刷新就好，之后正常使用很快

### Q3：免费额度够吗？

- ✅ 完全免费，不需要信用卡
- ✅ 无限静态文件 + API 请求
- ⚠️ 不活跃 15 分钟后暂停，下次访问需等待唤醒
- 个人作业/演示完全够用

### Q4：手机端能访问吗？

可以！Render 分配的域名在手机浏览器中直接打开即可，页面已做响应式适配。

### Q5：代码改了怎么更新？

1. 在电脑上修改代码
2. `git add .` → `git commit -m "更新"` → `git push`
3. Render 会自动检测到推送，重新部署（约 1-2 分钟）
